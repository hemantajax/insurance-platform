# Skills

Custom agent skills (slash commands). Each skill lives in its own subfolder with a `SKILL.md` file.

- `feature/` — Load, start, review, and complete features from `context/features/`
- `research/` — Generate documentation from research specs into `docs/`
- `cleanup/` — Housekeeping (unused imports, stale TODOs, orphaned files)
- `list-components/` — List React component files with descriptions
- `nx-workspace/`, `nx-run-tasks/`, `nx-generate/` — Nx monorepo helpers
